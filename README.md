# Untitled Jam Game

Built in a week, starting on March 15, 2024 for the [Chill Devs Chill Jam 7](https://itch.io/jam/chill-devs-chill-jam-7).

To play the game, run `cargo run` . Close the window or press `Esc` to exit.
